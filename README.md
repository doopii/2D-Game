# 🗝️ Treasure Hunting Game (v1.0)

Find keys, unlock doors, and get the treasure!

## 🎮 Gameplay
- Collect keys to unlock doors
- Grab boots to speed up
- Reach the chest to win

## 🕹️ Controls
- Move: W / A / S / D
